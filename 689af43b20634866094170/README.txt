Halyk RAG Challenge

Описания компонентов:
1) Dataset/ - папка содержит файлы с документами о компаниях (все данные являются публичными и взяты из kase.kz), которые вам предстоит предобработать и построить на их основе базу знаний для своего RAG пайплайна. В основном документы являются годовыми отчетами компаний за 2024 год, а также списками аффилированных лиц. Часть файлов представляет собой сканы, часть содержит сложные таблицы и диаграммы.
2) questions_public.xlsx - файл содержит список вопросов и форматы ответов.
3) submission_format.json - файл содержит шаблон для json файла с ответами.
3) answers_public.json - файл содержит ответы для вопросов из questions_public.xlsx в требуемом формате.


24-го августа будет опубликован приватный сет вопросов (датасет с документами остается тем же). На то, чтобы ответить на них, у вас будет 3 часа. До этого времени вам необходимо предобработать эти документы и подготовить свой RAG-пайплайн, чтобы успеть ответить на все вопросы из приватного сета.

Всем удачи!
